## What is APB Toolkit?

APB Toolkit is a free Windows desktop app that makes managing APB Reloaded configurations simple. Browse and apply community configs, upload your own, edit INI and localization files directly, and use a full suite of APB and system maintenance tools — all from one clean interface.

**[Download → apbtoolkit.com/download](https://apbtoolkit.com/download)**

---

## Features

### Config Management
- **Browse Configs** — browse and search community-uploaded configs with filters by category, sort by newest or most applied
- **My Configs** — manage your uploaded configs, view apply counts, delete entries
- **Upload Config** — drag and drop a config folder to upload it to the cloud. Files are validated and scanned before storage.
- **Apply Queue** — stage multiple configs and apply them to your APB installation in order with one click
- **Apply History** — full log of every config you've applied with timestamps

### Config Editor
- Edit APB INI files directly without leaving the toolkit
- Full localization file editor with inline color tag support
- Color picker for LOC color tags — handles both `<col:Name>` and `<Color:R= G= B=>` formats

### Utilities
Five tabs of APB and system tools:

| Tab | Tools |
|-----|-------|
| **APB** | Launch, Repair, Verify, Kill Process, Clear Cache/Logs, Open Folders, Create Shortcut, Reinstall, Uninstall |
| **GFAC** | Check Status, View Logs, Run Updater, Reinstall, Full Removal |
| **System** | High Performance Power Plan, Flush Shader Cache, Disable/Enable Xbox Game Bar, Task Manager, DxDiag |
| **Network** | Flush DNS, Release/Renew IP, Reset Winsock, Network Diagnostics |
| **Steam** | Open Steam, Open APB Page, Clear Download Cache, Verify/Reinstall/Uninstall APB via Steam |

### Settings
- APB installation path with auto-detection
- Discord OAuth and native account sign-in
- Cloud sync — settings and preferences synced across machines
- Run on Windows startup
- Anonymous usage statistics (opt-in)
- Automatic update checker and one-click installer

## Requirements

- Windows 10 or Windows 11 (64-bit)
- APB Reloaded installed via Steam
- Internet connection for cloud features

> .NET 8 Desktop Runtime is bundled inside the exe — no separate install required.

---

## Installation

1. Download `APBToolkit.exe` from **[apbtoolkit.com/download](https://apbtoolkit.com/download)**
2. Run it — no installer, no admin rights needed
3. Sign in with Discord or create an account
4. Set your APB installation path in Settings
